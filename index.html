2. SCOPE OF WORK

While this Budgetary Commercial Proposal provides an overview, a comprehensive technical proposal (ref:24M032/SP/170524) will be submitted alongside this document. The technical proposal contains detailed specifications and implementation plans.

2.1 System Components

  1. Network End Devices: These are the tracking devices assigned to workers in the plantation. They communicate worker locations and other relevant data.
  2. LoRaWAN Gateways: These gateways receive data from the end devices over long distances using LoRaWAN technology. They act as intermediaries, forwarding the data to the central monitoring system.
  3. Solar Power Systems: These systems power the gateways, ensuring uninterrupted operation in remote plantation areas where traditional power sources may not be available.
  4. API for Integration with Existing Plantation Central Monitoring Systems: This is the main hub where all data from the gateways is collected, processed, and displayed for plantation management to monitor worker activity in real time.

Key deliverables include:

  • 500 end devices for worker tracking
  • 2 LoRaWAN gateways
  • Solar-powered systems for gateways
  • System installation and commissioning
  • Training for 20 personnel

2.2 LoRaWAN Network Implementation

  1. Strategic placement of 2 gateways to ensure optimal coverage across the plantation.
  2. Use of long-range, low-power LoRaWAN protocol for efficient data transmission.
  3. Implementation of secure, bi-directional communication between end devices and gateways.

Implementation steps include:

  • Site survey and RF planning
  • Gateway installation at optimal locations
  • Network configuration and testing
  • Coverage verification

2.3 Integration with Existing Systems

  1. Provision of API for seamless integration with the plantation's current management systems
  2. Support for data exchange protocols compatible with common agricultural management software

Compliance with Malaysia's Personal Data Protection Act (PDPA) for worker tracking systems is outside the scope of this proposal. It is recommended that the end user ensure adherence to applicable laws and regulations regarding data privacy and security.
